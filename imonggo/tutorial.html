VIDEO TUTORIAL
The following tutorials on video will demonstrate how to set up and manage the powerful features of Imonggo. New videos will be added from time to time so be sure to check this page regularly.

Getting Started

Imonggo in 3 minutes

This 3-minute video will walk you through Imonggo’s basic concept of Stockroom, Store, and Office, and will show you how easy it is to capture your entire retail business cycle.

Upload Products & Quantity

This short video below will demonstrate how you can quickly upload your product definition and inventory quantity using Excel.

Increase Inventory Quantity

This video will demonstrate how to increase your inventory quantity by creating a “Receive from Supplier” transaction.

Advanced Features

How to Install Store v3

Imonggo Store v3 allows users to sell products even when internet connection is not available. This new version of is an update of Store v2, and is available for premium users only. This short video will show users how to install Store v3.

How to Install Store v2

This short video will demonstrate how to install Store v2, which allows premium users to sell products even when an internet connection is not available. Store v2 is faster, more powerful, and has more features than Imonggo’s first version of Store.

Managing Inventory with Tags

This short video will demonstrate how easy it is to use tags to classify and search inventories. It will also show you how to filter your sales reports using tags.

Tracking Salesman Sales

This video will demonstrate how to set up sales tracking for you to keep a tab of the performance of every salesperson.

Customer Membership Program

This short video will demonstrate how easy it is for Imonggo to implement a customer membership program to start rewarding your loyal customers with discounts and track their sales history.

Managing Users

This short video will demonstrate how to create additional users and assign them different roles for your Imonggo account.

Restricting Discounts

This short video will demonstrate how to restrict discounting for selected products.

Ignoring Inventory Tracking

This short video will demonstrate how to tag certain items as non-inventory tracking.

Setup Open Price

This short video will demonstrate how to set up selected products to have open price (i.e. allow cashier to modify price at the Store).

Selling with Decimal Quantity

This short video will demonstrate how to set up selected products to allow decimal or fractional quantity to be sold at the Store.
